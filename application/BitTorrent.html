BitTorrent 协议 / 客户端

2001年4月，美国的程序员Bram Cohen设计了BitTorrent协议。然后在2001年7月2日，他发布了第一个的BitTorrent的可执行版本。现在是由布莱姆·科恩的公司（BitTorrent的公司）保持维护。

μTorrent

2005年9月18日，μTorrent发布，它是一个非常强悍的软件，正因为它的强悍，2006年12月7日被BitTorrent, Inc.收购。2007年9月18日，BitTorrent 6.0 was released. Although previous versions of the BitTorrent client had been open source software, with version 6 it became closed. ，根据zhihu，v2ex网友的讨论 ，我猜测是这样，虽然BitTorrent设计了BitTorrent协议而且开发了第一款支持该协议的软件BitTorrent，但是人家μTorrent一经发布，就强大的无敌，所以把人家收购了，用了人家的代码，发现人家的代码太屌了，就干脆用了人家的代码，然后这么牛逼的代码，不能开源，干脆闭源之。
